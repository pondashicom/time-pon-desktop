# TIME-PON-DESKTOP

デスクトップ上に「透明・枠なし・常に最前面」のデジタルタイマーを表示する Electron アプリです。  
Overlay（表示）ウインドウと、Control（操作）ウインドウの **2ウインドウ構成**になっています。

- Overlay：透明・枠なし・最前面・クリック透過（文字だけ表示）
- Control：タイマー制御／見た目設定／カンペ送信

---

## 主な機能

### タイマー
- 表示形式：`HH:MM:SS` / `残り時間MMM:SS`の2パターン
- カウントダウン / カウントアップ
- 開始値（分）設定
- カウントダウン時は2段階の残り時間警告表示
- タイマーの表示 / 非表示
- 現在時刻表示の表示 / 非表示

### Overlay（透明表示）
- 透明背景、ウインドウ枠なし
- 常に最前面表示
- クリック透過
- 文字のフォント／サイズ／色を変更
- 表示先ディスプレイを選択（2画面運用向け）
- 表示位置とサイズも操作側で制御

### カンペ
- Control の入力欄に書いたテキストを Overlay に表示
- カンペの点滅機能あり
---

## 動作環境

- Windows
- Node.js / npm
- Electron（devDependencies）

---

## セットアップ & 起動

```bash
npm install
npm start
```

起動すると以下が開きます：

- **Overlay**：透明のタイマー表示（最前面・クリック透過）
- **Control**：操作画面（通常ウインドウ）

---

## 使い方

1. `npm start` で起動
2. Control で以下を設定
   - モード（カウントダウン/アップ）
   - 開始値（分）
   - Overlay の表示先ディスプレイ、フォント、サイズ、色
3. `スタート` で開始
4. カンペ欄に入力 → `送信` で Overlay に反映

---

## 仕様メモ

- タイマー内部は高精度で進みますが、表示は `HH:MM:SS` の整数秒に整形しています。
- 設定はユーザーデータ領域に保存し、次回起動時に復元します。

---

## 開発メモ

### ファイル構成（主要）
- `main.js`：ウインドウ生成、状態管理、タイマー制御、IPC
- `preload.js`：renderer への API 提供
- `overlay.html / overlay.js`：透明 Overlay 表示
- `control.html / control.js`：操作 UI
